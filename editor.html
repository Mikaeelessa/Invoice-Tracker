⚙️ GitHub Pages Setup

Connect your dashboard to GitHub so you can publish updates to your stakeholder viewer in one click.
🔑 How to get a Personal Access Token (2 minutes)
  1. Go to github.com → click your profile photo (top-right) → Settings
  2. Scroll to the bottom of the left sidebar → click Developer settings
  3. Click Personal access tokens → Tokens (classic)
  4. Click Generate new token (classic)
  5. Give it a name like payments-dashboard, set expiry to No expiration
  6. Tick the repo checkbox (top of scopes list) — this is the only permission needed
  7. Click Generate token → copy the token shown (starts with ghp_)
🔒 Stored only in your browser — never sent anywhere except GitHub's API.
